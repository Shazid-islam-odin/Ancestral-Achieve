# Ancestral Archive

A digital heritage museum web platform for preserving the oral histories, vintage photographs, and cinematic memories of a rural village community.

**All rights reserved © 2025 Shazid Islam Odin**

---

## About the project

Ancestral Archive is a front-end web application that allows community members to explore, contribute, and preserve local history through an elegant, accessible interface. The platform is designed with village elders and older generations in mind — large typography, simple audio controls, and a dignified, uncluttered aesthetic.

---

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Homepage — hero, recent archives, map preview, audio player |
| `pages/map.html` | Interactive Leaflet map with filterable location pins |
| `pages/timeline.html` | Horizontal scrollable timeline from 1900 to present |
| `pages/contribute.html` | Upload form for photographs, audio, and video |

---

## Features

- Warm earth-tone design with Playfair Display serif typography
- Interactive geographical map (Leaflet.js) with sepia tile styling
- Custom map markers by media type — photo, audio, video
- Horizontal draggable timeline with expandable milestone cards
- Waveform audio player with play/pause and progress tracking
- Drag-and-drop file upload with preview
- Fully responsive down to mobile
- Keyboard accessible throughout
- Zero build step — open `index.html` directly in any browser

---


## Design principles

- Serif display typography for a dignified, archival feel
- Earth tones: cream backgrounds, warm browns, muted ink
- Accessible font sizes and large touch targets for elder users
- No gradients, no clutter — every element earns its place
